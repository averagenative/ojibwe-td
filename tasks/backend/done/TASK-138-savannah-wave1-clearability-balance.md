## Description

Wave 1 on Savannah is currently not clearable with starting towers. The armored
bear creeps are too tanky for early-game DPS. The difficulty should escalate
linearly — wave 1 must be clearable with basic strategy.

This ties into the broader gear/enhance system: if we keep higher difficulty,
there needs to be a clear reason to do multiple runs (enhance tower stats via
meta-progression). Consider:
- Making "Enhance Tower" more prominent in menus
- Ensuring gear enhancement provides meaningful stat boosts
- Making it a grind to max out (replayability driver)
- Wave 1 should still be clearable even without enhancements

## Acceptance Criteria

- [ ] Audit Savannah wave 1 creep stats (HP, armor, speed) vs starting tower DPS
- [ ] Reduce armor bear HP/armor for wave 1 to be clearable with 2-3 basic towers
- [ ] Ensure linear difficulty escalation across first 5 waves
- [ ] Verify that gear-enhanced towers feel meaningfully stronger
- [ ] Consider surfacing "Enhance Tower" more prominently in meta menus
