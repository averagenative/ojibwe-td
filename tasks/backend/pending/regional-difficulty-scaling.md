## Description

All maps can currently be beaten with simple/singular tower setups. Later regions should
escalate difficulty so that simple strategies fail:

- Region 1 maps: beatable with basic arrow spam
- Region 2+: armored creeps force tower diversity
- Region 3+: mixed air+ground waves, status-immune creeps, boss mechanics that punish single-tower strategies
- Region 4+: requires full tower toolkit, upgrades, and meta-progression

This should also reflect in boss designs — later region bosses should have mechanics that
counter specific tower types.

## Acceptance Criteria

- [ ] Region 1 maps are accessible to new players with simple tower setups
- [ ] Region 2+ maps include enough armored/immune creeps that arrow-only fails
- [ ] Region 3+ maps include waves that counter single-tower strategies
- [ ] Boss designs escalate: later bosses have armor, immunity, or counter-mechanics
- [ ] Wave definitions updated with appropriate creep type distribution per region
- [ ] Difficulty curve feels smooth — no sudden brick walls between regions
- [ ] Playtested across all regions to verify scaling
- [ ] `npm run typecheck` clean

## Notes

- Depends on TASK-098 (tower role redesign) for armor mechanics to be meaningful
- Creep type distribution per wave is in `waves.json` or generated by WaveManager
- Boss definitions are in wave data — add region-appropriate abilities
- Consider: region-specific creep modifiers (e.g. all Region 3 creeps have +20% HP)
