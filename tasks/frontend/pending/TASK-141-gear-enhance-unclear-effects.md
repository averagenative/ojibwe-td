## Description

When enhancing gear, it's not clear what the enhancement actually does to the
gear's effects. Players need to see before/after stat changes or a clear
description of what each enhancement level provides.

## Acceptance Criteria

- [ ] Show current stat values and what they'll become after enhancement
- [ ] Use a before → after format (e.g. "Damage: 10 → 12")
- [ ] Show the enhancement cost clearly
- [ ] If enhancement is at max level, indicate that clearly
- [ ] Ensure the UI updates immediately after enhancing
