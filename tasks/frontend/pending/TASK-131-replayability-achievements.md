# Replayability Achievements — Lifetime Stats & Milestones

## Problem
The current achievement set (from TASK-110) may not have enough achievements that reward long-term play and replayability across many runs.

## Goal
Add achievements that track lifetime/cumulative stats to encourage repeated play.

## Requirements
Add achievements in these categories (suggested, adjust numbers for balance):

**Tower milestones:**
- Place X towers total (50, 200, 1000)
- Fully upgrade X towers of each type (5, 20, 50 per type)
- Have all 6 tower types placed simultaneously in one game

**Kill milestones:**
- Kill X creeps total (100, 500, 2000, 10000)
- Kill X bosses total (5, 20, 50)
- Kill X air creeps (50, 200)

**Economy milestones:**
- Earn X total gold across all runs (5000, 25000, 100000)
- Spend X crystals in the meta shop (500, 2500)
- Sell X towers (10, 50, 200)

**Run milestones:**
- Complete X runs (5, 20, 50, 100)
- Win on every map
- Win with every commander
- Win without losing a life (perfect run)

**Rush/challenge:**
- Rush X waves total (10, 50, 200)
- Complete a stage with only 1 tower type
- Reach wave 50 in endless mode

These should integrate with the existing achievements system from TASK-110. Track lifetime stats in SaveManager.
