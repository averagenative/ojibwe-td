# Commander Ability Hover Text with English Translation

## Problem
Commander abilities have Ojibwe names but no hover/tooltip text explaining what they do in English. Players don't know what each ability does without trial and error.

## Goal
Add on-hover tooltip text for commander abilities that includes an English translation/description of what the ability does.

## Requirements
- Show tooltip on hover (desktop) or long-press (mobile) for each commander ability
- Include the English translation of the Ojibwe ability name
- Include a brief description of the ability's mechanical effect
- Style consistently with existing UI tooltips
