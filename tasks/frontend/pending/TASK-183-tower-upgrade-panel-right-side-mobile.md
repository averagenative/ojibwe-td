## Description

The tower upgrade panel currently appears on the left side of the screen during gameplay. On mobile, it should be positioned on the right side for better thumb ergonomics. Additionally, all interactive elements within the upgrade panel (upgrade buttons, sell button, behavior toggles) need to meet minimum 44px tap target sizes on mobile.

## Acceptance Criteria

- [ ] Tower upgrade panel positioned on the right side of the screen on mobile
- [ ] All upgrade buttons meet 44px minimum tap target on mobile
- [ ] Sell button meets 44px minimum tap target on mobile
- [ ] Behavior toggle buttons meet 44px minimum tap target on mobile
- [ ] Panel doesn't overlap with HUD strip or tower selection bar
- [ ] Desktop layout unchanged (or improved)
